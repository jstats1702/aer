# Análisis Estadístico de Redes

En este repositorio se encuentran todos los materiales del curso **Análisis Estadístico de Redes**, ofrecido por el **Departamento de Estadística** de la **Universidad Nacional de Colombia**.

## Contenido

1. Conceptos preliminares.
2. Gestión de datos relacionales.
3. Conceptos fundamentales de teoría de grafos.
4. Visualización de datos relacionales.
5. Caracterización de vértices.
6. Caracterización de conectividad.
7. Agrupamiento.
8. Asortatividad.
9. Relaciones entre palabras.
10. Introducción a los modelos estadísticos para redes sociales.
11. Modelos de grafos aleatorios generalizado.
12. Modelos de grafos aleatorios exponenciales.
13. Modelos de bloques estocásticos.
14. Modelos de espacio latente de distancia.
15. Modelos para redes multicapa con efectos de sociabilidad.
16. Modelo para redes dinámicas con efectos de sociabilidad.
